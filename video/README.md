# Video (Remotion)

This project generates videos with Remotion.

## Get Started

Get started by running:

```bash
cd video
npm i
npm run dev
```

## Render

To render a video, run:

```bash
npx remotion render
```

## Task Runner Shortcuts (Repo Root)

```bash
task video:install
task video:dev
task video:render
task render
```
